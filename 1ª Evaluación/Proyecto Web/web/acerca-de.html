Este sitio web fue creado con el fin de mostrar mis conocimientos en el desarrollo web, y así poder obtener un trabajo en el área programación o diseño.

Para la realización de este proyecto, se partió del siguiente prototipo de fidelidad media diseñado con la herramienta Figma, mostrado a continuación:

Ver prototipo

El sitio web fue desarrollado con HTML5, CSS3 y JavaScript, también se utilizó una librería de JavaScript llamada JQuery, la cual nos permite realizar animaciones y efectos de manera más sencilla y rápida. Para el diseño del sitio web se utilizó la metodología de diseño llamada Desktop First, la cual consiste en diseñar el sitio web para dispositivos de escritorio y luego adaptarlo a dispositivos móviles.

En cuanto a la estructura del proyecto prevalecen:

  • Grid -> para la estructura principal
  • Flex -> para complementar a la estructura principal
  • @Media -> para adaptar el sitio web a todo tipo de dispositivos

Se ha hecho uso de etiquetas estructurales como:

  • header
  • nav
  • main
  • footer

En cuanto al CSS3 se ha hecho uso de:

  • Flex
  • Grid
  • Media Queries
  • Transiciones
  • Transformaciones
  • Animaciones
  • Keyframes
  • Clases
  • Herencia
  • Psuedo-clases

En cuanto al JavaScript y JQuery, se ha hecho uso de ellos para conseguir el menu "hamburguesa" y para que el texto se esconda debajo del header a medida que se vaya bajando en la página para que así, sea mas accesible.

En cuanto a la accesibilidad, se ha hecho uso de etiquetas semánticas para que el sitio web sea más accesible para personas con discapacidad visual, también se ha hecho uso de etiquetas alt en las imágenes para que así, las personas con discapacidad auditiva puedan saber de que trata la imagen.

Una vez terminado el proyecto se han pasado los archivos CSS por el validador para confirmar que no hay fallos.
